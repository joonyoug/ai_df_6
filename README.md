# AI를 활용한 총기사고 예방



## 📚 소개

이 프로젝트는 YOLO(Object Detection)를 활용하여 총기를 인식하고, 총기가 인식되었을 때 후속 조치를 취하는 시스템입니다. 이를 통해 총기사고를 예방하고, 긴급 상황에 신속히 대응할 수 있도록 설계되었습니다.
